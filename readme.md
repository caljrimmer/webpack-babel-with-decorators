Webpack and Babel setp with Decorators
----------------------------------------

**Setup**

We are using version 6.2.2 Node and 3.9.5 NPM. You can install them on Mac OSX with:

```unix
sh setup.sh
```

Then install the dependencies:

```unix
npm install
```

Finally start watching the ES6 files:

```unix
npm run watch
```
